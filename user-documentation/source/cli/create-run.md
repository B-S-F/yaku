# Create a run

In order to create a run you need to use the `yaku runs create <configId>` command.

```bash
yaku runs create 186
# {
#   "id": 2839,
#   "status": "pending",
#   "config": "https://yaku.bswf.tech/api/v1/namespaces/1/configs/148",
#   "creationTime": "2023-03-01T17:48:05.050Z"
# }
```
